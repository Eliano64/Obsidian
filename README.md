# Obsidian Study Notes

Personal study notes as an Obsidian vault that are across operating systems, computer organization, networking, data structures and algorithms, programming languages, and tools.

Some notes embed interactive HTML slides via the **Local HTML Card** plugin. You must install it before opening this vault, otherwise slides will not render.

**Plugin:** [local-html-card-plugin](https://github.com/Eliano64/local-html-card-plugin)

## Setup

```bash
git clone https://github.com/Eliano64/Obsidian.git
cd Obsidian
# Copy the plugin into the vault:
mkdir -p .obsidian/plugins/local-html-card
git clone https://github.com/Eliano64/local-html-card-plugin.git .obsidian/plugins/local-html-card
```

Open the folder as an Obsidian vault and enable **Local HTML Card** in Settings → Community plugins → Installed plugins.
