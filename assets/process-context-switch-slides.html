1. A 正在 CPU 上运行

当前 CPU 寄存器现场属于进程 A；B 在就绪队列中等待。

切换前

2. 保存 A 的运行现场

先把 CPU 中的 PC、PSW、通用寄存器保存到 PCB A。

CPU -> PCB A

3. A 离开 CPU 并进入队列

保存现场后，修改 PCB A 的状态和队列位置。

PCB A 入队

4. 调度程序选择 B

从就绪队列中选中 B，准备把 B 的现场装入 CPU。

选择 PCB B

5. 恢复 B 的运行现场

把 PCB B 中保存的 PC、PSW、通用寄存器写回 CPU，B 从上次停下的位置继续执行。

PCB B -> CPU
