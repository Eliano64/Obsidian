单链表翻转区间 [left, right]

示例：head → 1 → 2 → 3 → 4 → 5 → 6，翻转 [2,5]

1. 找到区间前驱和区间首结点

before 停在第 left-1 个结点，first 指向第 left 个结点。

before 不再移动；first 最终会成为翻转区间的尾结点。

2. 把 first 后面的结点移到区间最前面

第一轮取下结点 3，将它插到 before 与结点 2 之间。

moved = first->next; first->next = moved->next; moved->next = before->next; before->next = moved;

3. 重复同一组改链

结点 3、4 已依次移到前端；first 仍指向结点 2，下一次移动结点 5。

循环执行 right-left 次。每一轮只搬动一个结点，已经翻转好的前半段不会被破坏。

4. 区间翻转完成

区间外的结点保持原顺序，原区间首结点 2 成为区间尾结点并接回结点 6。

结果：head → 1 → 5 → 4 → 3 → 2 → 6。时间复杂度为 O(n)，额外空间为 O(1)。
