# 网络、互联网、因特网

**网络**是若干结点和链路互连形成的系统。结点可以是主机、路由器、交换机等设备，链路用于在结点之间传送信号或数据。

**互联网**是若干网络通过路由器互连形成的更大网络。这里的“互”强调的是网络与网络之间的互连。

**因特网**是当今世界上最大的互联网。它不是某一个局域网，也不是某一种具体设备，而是由大量异构网络、路由器、主机和协议共同构成的全球性互联网。

![900](../assets/network-internet-relationship.svg)

>[!note] 大小写容易混淆
>英文中，`internet` 可以泛指“互联网”，即多个网络互连形成的网络；`Internet` 通常专指全球因特网。

# 计算机网络的组成

从组成上看，计算机网络至少包含：

| 组成 | 作用 |
| --- | --- |
| 主机 | 运行用户程序，产生和接收数据，如 PC、手机、服务器 |
| 通信链路 | 连接网络结点，承载信号传输 |
| 交换设备 | 在网络内部转发数据，如交换机 |
| 路由器 | 连接不同网络，并在网络之间转发分组 |
| 协议 | 规定通信双方如何组织、发送、接收和解释数据 |

在因特网中，还常把组成分成**边缘部分**和**核心部分**。

![900](../assets/internet-edge-core.svg)

边缘部分由用户直接使用的主机构成，例如个人电脑、智能手机、服务器、网络摄像头等。它们运行应用程序，直接产生和消费网络数据。

核心部分由大量异构网络和互连这些网络的路由器构成。它不直接面对最终用户应用，而是为边缘主机之间的通信提供连通性和数据交换能力。

# 计算机网络的定义

早期可以把计算机网络理解为“一些互连的、自治的计算机的集合”：

- **互连**：计算机之间能够进行数据通信。
- **自治**：每台计算机有自己的软硬件，能够独立运行。

更适合现代网络的说法是：计算机网络由一些通用的、可编程的硬件互连而成，这些硬件可以传送多种类型的数据，并支持各种网络应用。

这个定义强调两点：

- 网络连接的设备不限于传统计算机，也包括手机、传感器、智能家电等带有处理能力的设备。
- 计算机网络不是只为传送某一种信号而存在，而是能在数据传送基础上支持不断变化的应用。

# 计算机网络的功能

计算机网络的基本功能可以概括为：

| 功能 | 含义 |
| --- | --- |
| 数据通信 | 在不同主机、进程或用户之间传送数据 |
| 资源共享 | 共享硬件、软件和数据资源，如打印机、文件、数据库 |
| 分布式处理 | 多台计算机协同完成任务 |
| 提高可靠性 | 某些结点或链路故障时，可以通过冗余路径或备用系统继续工作 |
| 负载均衡 | 把请求或任务分散到多台设备上处理 |

其中，数据通信是最基础的功能。没有数据通信，其他功能都无法建立。

# 计算机网络的分类

![900](../assets/computer-network-classification.svg)

## 按覆盖范围分类

| 类型 | 英文 | 覆盖范围 | 典型场景 |
| --- | --- | --- | --- |
| 个域网 | PAN | 约 10 m | 蓝牙耳机、鼠标、个人设备互连 |
| 局域网 | LAN | 约 1 km | 宿舍、楼宇、校园、企业内部网络 |
| 城域网 | MAN | 约 5-50 km | 城市骨干网、城区范围网络 |
| 广域网 | WAN | 几十千米到几千千米 | 跨地区、跨国家的大范围网络 |

这组分类看的是网络覆盖的地理范围。范围越大，通常越依赖运营商、骨干链路和路由基础设施。

## 按使用者分类

| 类型 | 含义 |
| --- | --- |
| 公用网 | 面向公众提供服务，通常由电信公司或网络运营商建设和运营 |
| 专用网 | 为某个单位或部门的特定业务建设，不向外部公众开放 |

专用网不是说不能使用公用网络的线路，而是强调服务对象和管理边界。

## 其他分类角度

| 分类角度 | 常见类别 |
| --- | --- |
| 传输介质 | 有线网络、无线网络 |
| 拓扑结构 | 总线型、星型、环型、网状型 |
| 交换方式 | 电路交换、报文交换、分组交换 |
| 传输技术 | 点对点网络、广播式网络 |

分类角度不同，得到的类别也不同。同一个网络可以同时属于多个分类。例如，一个校园 Wi-Fi 网络可以是局域网、无线网络、星型或扩展星型结构的一部分，也可能通过分组交换接入因特网。
